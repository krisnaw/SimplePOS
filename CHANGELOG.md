# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-03

### Added
- Initial project structure with Electron, React, and TypeScript.
- Integrated TypeORM with SQLite for local data storage.
- Implemented core database schema for Car Repair Shop management:
    - Users and Authentication.
    - Customers and Vehicles.
    - Products (Parts) and Services.
    - Work Orders and Invoices.
- Setup Shadcn UI components for a modern look and feel.
- Developed main application workspaces:
    - Dashboard Overview.
    - Sales/Work Order Management.
    - Inventory Management.
    - Customer/User Management.
    - Reports.
- Added detailed documentation (ERD, Architecture, Sample Data).
- Configured Vite for fast frontend development.
- Configured Electron-Vite HMR for smooth developer experience.
