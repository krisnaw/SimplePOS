# SimplePOS — Role Permissions

## Roles

- **Admin** — full access to all features including settings, reports, and inventory management
- **Cashier** — limited to day-to-day operations: work orders, customers, and checkout

## Permissions Matrix

| Feature | Admin | Cashier |
|---|---|---|
| Dashboard | ✅ Full | ✅ Limited |
| Work Orders | ✅ | ✅ |
| Customers & Vehicles | ✅ | ✅ |
| Services management | ✅ | ❌ |
| Inventory / Products | ✅ | ❌ (view only in WO) |
| Reports | ✅ | ❌ |
| User Management | ✅ | ❌ |
| Shop Settings | ✅ | ❌ |
